Detailed Description

template<Size N>
class Sandals::SemiExplicit< N >

Class container for the system of explicit ODEs.

Class container for the system of semi-explicit ordinary differential equations (ODEs) of the type \( \mathbf{M}(\mathbf{x}, t)\mathbf{x}^{\prime} = \mathbf{r}(\mathbf{x}, t) \).
