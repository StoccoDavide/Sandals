Detailed Description

template<Size N, Size H>
class Sandals::SemiExplicit< N, H >

Class container for the system of explicit ODEs/DAEs.

Class container for the system of semi-explicit ordinary differential equations (ODEs) or differential algebraic equations (DAEs)of the type \( \mathbf{M}(\mathbf{x}, t) \mathbf{x}^{\prime} = \mathbf{r}(\mathbf{x}, t) \), with invariants manifold \( \mathbf{h}( \mathbf{x}, t) = \mathbf{0} \).
