Detailed Description

template<Size N, Size M>
class Sandals::Linear< N, M >

Class container for the system of linear ODEs/DAEs.

Class container for the system of linear ordinary differential equations (ODEs) or differential algebraic equations (DAEs) of the type \( \mathbf{E}(t)\mathbf{x}^{\prime} = \mathbf{A}(t) \mathbf{x} + \mathbf{f}(t) \), with invariants manifold \( \mathbf{h}(\mathbf{x}, t) = \mathbf{0} \).
