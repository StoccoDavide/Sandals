Detailed Description

template<Size N>
class Sandals::Linear< N >

Class container for the system of linear ODEs/DAEs.

Class container for the system of linear ordinary differential equations (ODEs) or differential algebraic equations (DAEs) of the type \( \mathbf{E}(t)\mathbf{x}^{\prime} = \mathbf{A}(t) \mathbf{x} + \mathbf{f}(t) \).
