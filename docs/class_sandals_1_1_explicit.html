Detailed Description

template<unsigned N>
class Sandals::Explicit< N >

Class container for the system of explicit Ordinary Differential Equations (ODEs) of the type \( \mathbf{x}^{\prime} = \mathbf{f}(\mathbf{x}, t) \).
