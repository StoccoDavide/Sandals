Detailed Description

template<Size N>
class Sandals::Explicit< N >

Class container for the system of explicit ODEs.

Class container for the system of explicit ordinary differential equations (ODEs) of the type \( \mathbf{x}^{\prime} = \mathbf{f}(\mathbf{x}, t) \).

Evaluate the Jacobian of the ODE system function \( \mathbf{F}(\mathbf{x}, \mathbf{x}^{\prime}, t) \) with respect to the states \( \mathbf{x} \)
