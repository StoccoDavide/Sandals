Detailed Description

template<Size N>
class Sandals::Implicit< N >

Class container for the system of implicit ODEs/DAEs.

Class container for the system of inplicit ordinary differential equations (ODEs) or differential algebraic equations (DAEs) of the type \( \mathbf{F}(\mathbf{x}, \mathbf{x}^{\prime}, t) = \mathbf{0} \).
