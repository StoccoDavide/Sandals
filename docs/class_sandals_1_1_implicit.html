Detailed Description

template<unsigned N>
class Sandals::Implicit< N >

Class container for the system of implicit Ordinary Differential Equations (ODEs) of the type \( \mathbf{F}(\mathbf{x}, \mathbf{x}^{\prime}, t) = \mathbf{0} \).
